### Java 注释

注释可用于解释Java代码并使其更具可读性。在测试备用代码时，它也可以用于阻止执行。
单行注释以两个正斜杠（//）开头。

本示例在一行代码之前使用单行注释：
```
// This is a comment
System.out.println("Hello World");
```

本示例在代码行的末尾使用单行注释：
```
System.out.println("Hello World"); // This is a comment
```

### Java多行注释
多行注释以开头/*和结尾*/。

/*和之间的任何文本*/都将被Java忽略。

本示例使用多行注释（注释块）来解释代码：
```
/* The code below will print the words Hello World
to the screen, and it is amazing */
System.out.println("Hello World");
```
